Student Performance Analysis Using Python
Project Objective
Analyze student academic performance data to identify trends, subject-wise strengths and weaknesses, and performance gaps related to attendance and demographics.
Dataset
The dataset contains student-level academic records, including:
Math, English, and Science scores
Attendance information
Demographic attributes
The dataset is used for academic learning and analysis purposes.
Tools Used
Python
Pandas for data manipulation
Matplotlib for visualization
Project Workflow
Loaded and inspected the dataset
Cleaned data by handling missing and invalid values
Engineered new features such as total score, average score, and performance groups
Performed descriptive statistical analysis
Analyzed performance gaps across subjects, attendance levels, and demographics
Visualized trends using histograms, bar charts, and scatter plots

Conclusion
This project demonstrates practical data analysis using Python, with a focus on cleaning data, extracting insights, and presenting results clearly. It reflects foundational skills required for data analytics and machine learning internships.
